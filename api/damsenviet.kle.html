class Key¶

Class for storing a KLE Key.

Variables

  • x (Decimal) – x position of the main shape, defaults to Decimal(0.0)

  • y (Decimal) – y position of the main shape, defaults to Decimal(0.0)

  • width (Decimal) – width of the main shape, defaults to Decimal(1.0)

  • height (Decimal) – height of the main shape, defaults to Decimal(1.0)

  • x2 (Decimal) – x position offset of the secondary shape, defaults to Decimal(0.0)

  • y2 (Decimal) – y position offset of the secondary shape, defaults to Decimal(0.0)

  • width2 (Decimal) – width of the secondary shape, defaults to Decimal(1.0)

  • height2 (Decimal) – height of hte secondary shape, defaults to Decimal(1.0)

  • rotation_x (Decimal) – x position of the origin of rotation, defaults to Decimal(0.0)

  • rotation_y (Decimal) – y position of the origin of rotation, defaults to Decimal(0.0)

  • rotation_angle (Decimal) – rotation angle about the origin in degrees, defaults to Decimal(0.0)

  • color (str) – fill color, defaults to “#cccccc”

  • text_labels (List[str]) – text labels , defaults to [“” for i in range(12)]

  • text_colors (List[Union[str, None]]) – text colors of the labels, defaults to [None for i in range(12)]

  • text_size (List[Union[int, float, None]]) – text sizes of the labels, defaults to [None for i in range(12)]

  • default_text_color (str) – default text color, defaults to “#000000”

  • default_text_size (Union[int, float]) – default text size, defaults to 3

  • decal (bool) – whether the key is decorative, defaults to False

  • ghost (bool) – whether the key is invisible, defaults to False

  • stepped (bool) – whether the key is stepped, defaults to False

  • nub (bool) – whether the key is nubbed, defaults to False

  • profile (str) – the profile of the key, defaults to “”

  • switch_mount (str) – switch mount, defaults to “”

  • switch_brand (str) – switch brand, defaults to “”

  • switch_type (str) – switch type, defaults to “”
